## 03-hello-world-http

### Description:

A C example that implements a very basic TCP/IP stack and web server. Based on [minIP](https://github.com/IanSeyler/minIP)

### Prerequisites:

- `libBareMetal.c` and `libBareMetal.h`. The `build.sh` script will download them.

### Compile:
```sh
./build.sh
```
